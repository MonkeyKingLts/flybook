# Flybook 设计规范

> 设计风格：**飞书（Lark）简化版** — 干净、专业、高效的企业协作 SaaS  
> 用途：Stitch 生成设计图、前端开发、简历作品集

---

## 1. 产品定位

| 项 | 说明 |
|----|------|
| 产品名 | Flybook |
| 类型 | 多租户任务 / 项目管理系统 |
| 参考 | 飞书任务 + 飞书项目（轻量版） |
| 平台 | 桌面 Web 优先（1440px），后续适配 1280px |
| 语言 | 简体中文为主 |

---

## 2. 设计原则（飞书风）

1. **效率优先**：信息密度适中，减少装饰，一屏看到关键内容  
2. **层次清晰**：背景 / 卡片 / 浮层三级分明，用边框而非重阴影分区  
3. **色彩克制**：大面积白与浅灰，品牌色只用于主操作与关键状态  
4. **圆角统一**：卡片 8px，按钮 6px，标签 4px，头像圆形  
5. **反馈明确**：hover 有背景变化，操作后有 Toast，危险操作用红色

## 5. 间距与圆角

### 5.1 间距（4px 基准）

| Token | 值 | 用途 |
|-------|-----|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 卡片内紧凑间距 |
| md | 12px | 表单项间距 |
| lg | 16px | 卡片 padding |
| xl | 24px | 区块间距 |
| 2xl | 32px | 页面边距 |

### 5.2 圆角

| Token | 值 | 用途 |
|-------|-----|------|
| sm | 4px | 标签、小按钮 |
| md | 6px | 按钮、输入框 |
| lg | 8px | 卡片、弹窗 |
| full | 50% | 头像 |

### 5.3 阴影（飞书风：轻阴影）

| 层级 | 值 | 用途 |
|------|-----|------|
| Card | `0 1px 2px rgba(0,0,0,0.06)` | 卡片默认 |
| Dropdown | `0 4px 12px rgba(0,0,0,0.1)` | 下拉、Popover |
| Modal | `0 8px 24px rgba(0,0,0,0.12)` | 弹窗 |
| Drawer | `-4px 0 16px rgba(0,0,0,0.08)` | 右侧抽屉 |

## 6. 布局结构

### 6.1 全局框架

```
┌──────────────────────────────────────────────────────────────┐
│ 顶栏 56px  白底 + 底边框                                       │
├────────────┬─────────────────────────────────────────────────┤
│ 侧栏 220px │ 主内容区                                         │
│ 白底       │ 背景 #F5F6F7                                     │
│            │ 内边距 24px                                      │
│            │                                                  │

```

### 6.2 顶栏（56px）

- 左：组织切换器（Logo + 组织名 + 下拉箭头）

| 图标 | 文案 | 路由 |
|------|------|------|
| 首页 | 工作台 | `/dashboard` |
| 任务 | 我的任务 | `/my-tasks` |
| 文件夹 | 项目 | `/projects` |
| 设置 | 组织设置 | `/org/settings` |

## 7. 核心组件规范

### 7.1 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 背景 `#3370FF`，白字，高 32px，padding 0 16px |
| 次按钮 | 白底，边框 `#E5E6EB`，文字 `#1F2329` |
| 文字按钮 | 无底色，文字 `#3370FF` |
| 危险按钮 | 白底红字或红底白字，仅删除等场景 |

### 7.2 输入框

- 高 32px，边框 `#E5E6EB`，圆角 6px  
- 聚焦：边框 `#3370FF`，外发光 `0 0 0 2px #E8F0FF`  
- 占位符：`#8F959E`

### 7.3 标签 Tag

- 高 22px，padding 0 8px，圆角 4px  
- 浅色底 + 深色字（见状态色表）  
- 不加描边

### 7.4 任务卡片（看板）

```
┌─────────────────────────────┐
│ ▌ FB-12                     │  ← 左侧 3px 优先级色条
│   完成登录页 UI 设计          │
│   📅 3月15日    👤 头像       │
└─────────────────────────────┘
```

- 白底，边框 `#E5E6EB`，圆角 8px，padding 12px  
- hover：边框 `#3370FF`，轻阴影  
- 拖拽中：阴影加深，轻微旋转 1°

### 7.5 看板列

- 列宽 280px，列间距 12px  
- 列头：状态名 + 数量 `(5)`，字号 14px 600  
- 列背景：`#F5F6F7`，圆角 8px，padding 12px  
- 列底「+ 添加任务」：虚线边框按钮

### 7.6 任务详情抽屉

- 宽 480px，从右侧滑入  
- 顶栏：状态下拉 + 更多菜单 + 关闭  
- 属性行：左标签 80px `#646A73`，右内容  
- 评论区：时间线样式，头像 + 姓名 + 内容 + 时间

### 7.7 表格（列表视图）

- 表头：背景 `#F5F6F7`，文字 `#646A73`，高 40px  
- 行高 48px，hover 背景 `#F5F6F7`  
- 分割线 `#F0F1F3`

### 7.8 空状态

- 居中插画（线条风格，灰蓝色）  
- 主文案 14px `#646A73`  
- 下方蓝色主按钮引导操作

## 10. 动效（简要）

| 场景 | 规格 |
|------|------|
| 页面切换 | 无动画或 150ms fade |
| 抽屉打开 | 右侧滑入 250ms ease-out |
| Modal | 居中缩放 200ms |
| 拖拽卡片 | 即时跟随，落点弹性 150ms |
| Toast | 顶部滑入，3s 自动消失 |

## 11. 飞书风 Do / Don't

### Do

- 用浅灰背景 + 白色卡片分区  
- 状态用浅色底标签  
- 主操作只有一个蓝色按钮  
- 信息行对齐、间距一致  
- 中文文案简洁（「新建任务」而非「Create New Task」）

### Don't

- 不要用渐变背景、玻璃拟态、霓虹色  
- 不要过重阴影或立体按钮  
- 不要一页超过 2 种强调色  
- 不要用英文界面  
- 不要过度留白导致信息过少
